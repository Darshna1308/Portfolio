# 🌟 Personal Portfolio Website
A responsive and modern personal portfolio website built using **HTML, CSS, and JavaScript**.  
This project showcases my skills, projects, resume, and contact information in a clean and interactive UI.

## ✨ Features

- 🎨 Clean, modern, and aesthetic UI (pink + golden cinematic theme)
- 📱 Fully responsive design (mobile + desktop friendly)
- 🧭 Smooth scrolling navigation
- 💫 Hover animations and interactive UI elements
- 🧑 About section with professional profile summary
- 💼 Projects section with real-world AI/ML projects
- 📄 Resume download/view option
- 📬 Contact information section
## 🛠️ Tech Stack

- HTML5
- CSS3 (Flexbox + Grid + Glassmorphism)
- JavaScript (Basic DOM interaction)
